# Cookie Consent Bar

A lightweight, GDPR-friendly cookie consent banner with accept/decline and a
remembered choice — no bloat, no external calls. Part of the
[Zub Plugin Factory](https://github.com/ZubeidHendricks/wp-plugin-factory).

## What it does

Shows a customisable consent banner, remembers the choice in a cookie for 180
days, and fires a `zcc:consent` JS event so developers can gate their own
scripts. Optional script-blocking (true GDPR mode) is Pro.

## Settings

**Settings → Cookie Consent Bar** — enable, message, buttons, policy URL,
colour, script blocking (Pro).

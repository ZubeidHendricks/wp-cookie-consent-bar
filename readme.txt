A lightweight, GDPR-friendly cookie consent banner with accept/decline and a remembered choice.

== Description ==

A clean cookie consent banner that loads fast, makes no external calls, and
remembers the visitor's choice for six months. Customise the message, buttons,
privacy-policy link and colours. Fires a JavaScript event on consent so you can
hook your own scripts.

**Free features**

* Accept / decline with remembered choice (cookie, 180 days).
* Custom message, buttons, colours and policy link.
* `zcc:consent` JS event for developers.
* No external requests, tiny footprint.

**Pro features**

* Block analytics/marketing scripts until consent (true GDPR mode).
* Geo-targeting (EU only).
* Consent logging.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate — the banner shows immediately.
3. Customise under Settings → Cookie Consent Bar.

== Frequently Asked Questions ==

= Does this make my site fully GDPR compliant? =

It provides the consent banner and remembers the choice. Truly blocking scripts
before consent is a Pro feature. Compliance also depends on your wider setup.
